# ScaffoldsDifPerfPaper_2023

This repository contains all the Python scripts developed to process the MRI data from the paper "Leveraging magnetic resonance imaging to assess biocompatible scaffolds diffusion and flow-driven perfusion for lab-on-a-chip systems" (DOI: https://doi.org/10.1038/s41598-025-28465-w). 

The MRI data generated and processed for this study can be found at https://doi.org/10.34810/data1028. 

For any doubts, please contact David Gomez-Cabeza (dgomez@ibecbarcelona.eu).
